Dropbox to Markdown

Paste Dropbox share links and convert them into Markdown image syntax in one step.

About rlkey: New-format links (/scl/) require an rlkey query parameter. This tool keeps rlkey and replaces dl=0 with raw=1.
Show usage examples
Input (legacy /s/ format)
